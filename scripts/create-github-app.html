Create Able Terraform GitHub App

Create New GitHub App

Step 1: Fill in the form

  • GitHub App name: Able Terraform
  • Homepage URL: https://github.com/mpotter/able
  • Webhook: Uncheck "Active"

Step 2: Set permissions

Repository permissions:
  • Administration: Read and write
  • Contents: Read-only
  • Environments: Read and write
  • Metadata: Read-only (auto-selected)
  • Pull requests: Read and write
  • Actions: Read and write

Step 3: Create and install

  1. Click "Create GitHub App"
  2. Click "Install App" in the sidebar
  3. Select "Only select repositories" â†’ choose able
  4. Click "Install"

Step 4: Get credentials

  1. Go to the App's General settings
  2. Copy the App ID (near the top)
  3. Scroll to "Private keys" and click "Generate a private key"

Step 5: Set secrets

gh secret set GH_APP_ID
# Paste the App ID when prompted

gh secret set GH_APP_PRIVATE_KEY < ~/Downloads/able-terraform.*.pem

Step 6: Verify

./scripts/setup.sh
